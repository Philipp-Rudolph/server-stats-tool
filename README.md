# Server Stats CLI Tool
## Description

A command-line tool to monitor and display server statistics.

## Features

- CPU usage
- Memory usage
- Disk usage
- Network statistics

## Installation

```sh
git clone git@github.com:Philipp-Rudolph/server-stats-tool.git
```

## Usage

```sh
cd server-stats
./server-stats.sh
```
